# Project Scope

This file explains what the current v1 does and what it deliberately does not try to do.

## Included now

- Physical profiling
- Deterministic semantic inference
- Deterministic dictionary outputs (`.md`, `.csv`, `.json`)
- Optional config overrides
- `suggested_overrides.yaml` review workflow
- Bounded agent mode
- `agent_trace.json`
- `agent_report.md`
- Optional LLM description suggestions
- `llm_safe_summary.json`
- `llm_description_suggestions.json`
- `llm_description_suggestions.md`

## Still not included

- Formal sensitive data classification
- Data catalog publishing
- Framework-based orchestration
- Automatic replacement of governed business definitions
- Web UI
- Database-backed metadata repository
- Autonomous open-ended agent behaviour

# AGENTS

This repository is an agent project, but there is **no agent mode yet** in Milestone 0/1.

## Why the name still includes "agent"

The long-term roadmap includes a bounded orchestration layer that will consume deterministic profiling evidence to assist data dictionary workflows.

## Where agent mode will be added

Future milestones will introduce explicit agent orchestration modules. They are intentionally excluded from this milestone.

## Current authority

Deterministic profiling output remains authoritative. Any future agent behavior must stay grounded in these trace artifacts.
